## Изученные теги и элементы

### Основные теги

- Заголовки: `<h1>` - `<h6>`
- Абзац: `<p>`
- Выделение: `<strong>`, `<em>`
- Ссылки: `<a href="">`
- Изображения: `<img src="" alt="">`

### Списки

- Нумерованный: `<ol>`
- Маркированный: `<ul>`
- Элемент списка: `<li>`

### Формы

- Форма: `<form>`
- Поле ввода: `<input>`
- Кнопка: `<button>`
- Текстовая область: `<textarea>`

### Семантические теги

- `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>`
